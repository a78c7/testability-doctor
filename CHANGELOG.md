# Changelog

## v0.1.0

- Initial open-source release.
- Added static repository stack detection for Node, Python, Go, Rust, Swift, Java, docs-only, and unknown repositories.
- Added package manager, lockfile, test command, external environment, monorepo, and size checks.
- Added Markdown and JSON reports.
- Added config support and `init-config`.
- Added examples, unit tests, GitHub Actions workflow, and release packaging.
